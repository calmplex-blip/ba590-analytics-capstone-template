# BA590 Masters Project in Data Analytics

This individual capstone develops one evidence-based analytics solution through
four cumulative Classroom 50 milestones. Canvas is the official submission,
rubric, feedback, and grade record.

## Evidence standard

Connect every material business recommendation to authorized data, reproducible
analysis, captured output, evaluation, responsible-data-use boundaries, and a
decision record. The final project must run from a clean environment and disclose
limitations, monitoring, recourse, and the condition that would require the
recommendation to change.

## Required paths

- `problem_statement.md`, `evidence_review.md`, and `project_charter.md`
- `data_contract.md` and `method_decision.md`
- `notebooks/feasibility_baseline.ipynb`, `notebooks/data_pipeline.ipynb`,
  `notebooks/analysis_baseline.ipynb`, and `notebooks/final_analysis.ipynb`
- `evaluation_report.md`, `responsible_data_use.md`, and `report_draft.md`
- `final_report.md`, `presentation/final_defense.md`, and `handoff_runbook.md`

## Single-student participation

The work is individual. If no peer is enrolled, perform stakeholder, analyst,
data steward, reviewer, decision owner, and operator perspectives sequentially.
Preserve each challenge, response, revision, and remaining limit; use instructor
or simulated-stakeholder feedback where review is required.
